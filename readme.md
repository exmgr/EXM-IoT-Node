## Features

- Energy autonomous - 2W solar panel and x2 18650 batteries
- Communication autonomous - GSM/LTE/NBIoT/WiFi/LoRa/BLE/GPS
- Multiple sensor protocols - SDI-12/UART/RF/PWM

- Bosch BME 280 (temperature, humidity, pressure)
